# 格拉哥里字母翻译器

这是一个非常简单易用的网页小工具。它的主要作用是让你在**我们常用的英文字母和古老的格拉哥里字母（也是游戏《巫师》系列中使用的文字）** 之间进行互相翻译。

你不需要安装任何软件，只需要用浏览器双击打开 `glagolitique_complate.html` 这个文件就可以直接使用了。

注意：因为Q和X有映射的问题所以尽量不要用带这两个字母的句子。

## ✨ 这个网页有什么特性？

为了让使用体验更好，我在这个网页中加入了以下特性：

- **彻底告别“乱码”**
  - 很多时候，把奇怪的符号发给别人，或者在自己电脑上打开，会变成一个个方块（乱码）。这个网页**内部自带了专门的古文字体**。纯离线使用，不需要额外下载资源，只要打开网页，字形就能完美、清晰地显示出来。
- 多模式支持
  - 该工具支持巫师二使用的字体和巫师三使用的字体。可以点击切换。

## 🛠️ 我做了哪些功能？

这个网页主要包含了以下几个核心功能：

1. **实时双向翻译**
   - **英文 ➡️ 神秘字母**：在左边的框里输入平时用的英文字母，右边就会立刻变成对应的格拉哥里字母。
   - **神秘字母 ➡️ 英文**：点击中间的“左右箭头”按钮，就可以反过来，把你看到的神秘符号翻译成你能看懂的英文。
2. **自带“屏幕软键盘”（全矩阵面板）**
   - 网页下半部分有一个像键盘一样的面板。因为普通人的电脑键盘上打不出这些神秘符号，所以你可以**直接用鼠标点击面板上的方块**，它就会自动把符号输入到框里，非常直观。
3. **一键复制与清空**
   - 写好一段神秘文字后，点击 **“复制”** 按钮，就可以直接粘贴发给你的朋友。
   - 点击 **“清空”** 按钮，就能瞬间把输入框打扫干净，准备下一次翻译。
4. **一键生成并导出图片**
   - **痛点解决**：《巫师三》中的使用的是方体字母，《巫师二》使用的是圆体字母。如果在自己的电脑上渲染，需要下载特定字体。我做了三种导出模式，方便分享和绘图参考。
   - **如何使用**：点击界面上的**“导出”**按钮，即可实现，图片模式只导出转换后的文本， 支持透明底和自定义尺寸，text模式和pdf模式导出转换后的文本和原始文本。
5. **内置新手帮助**
   - 点击右上角的“帮助 / 关于”按钮，会弹出一个说明书，告诉你这些文字的来源（比如《巫师》游戏）以及一些发音对照规则。

## 2. 格拉哥里字母的历史定位

Glagolitic script 是已知最早用于书写斯拉夫语言的字母体系。学界普遍认为，该系统由拜占庭传教士 Saint Cyril 与 Saint Methodius 在9世纪中期（约公元863年前后）为大摩拉维亚传教活动所创制，用于翻译宗教典籍并记录 Old Church Slavonic。

从语言学角度看，格拉哥里字母是一套高度表音化（phonemic）的书写系统。其设计目标是较为准确地表示斯拉夫语族的音位结构，包括多种咝音（sibilants）、腭化辅音以及鼻化元音等特征音位。

在字形设计上，格拉哥里字母具有高度原创性，与同时期的希腊或拉丁字母在视觉形态上差异明显。但在更宏观的体系结构层面，其字母顺序、命名体系以及部分结构原则仍受到希腊字母传统的影响。

## 3. 理论基础

### 3.1 理论基础 I：格拉哥里字母与西里尔文的字母对应关系

要建立格拉哥里字母与现代英语之间的转换路径，通常需要借助 Cyrillic script 作为中间层级。

早期西里尔字母体系形成于保加利亚第一帝国时期的文化中心，例如 Preslav Literary School。学界普遍认为，该字母体系在结构上吸收了希腊安色尔体（Greek Uncial）的字形特征，并结合斯拉夫语音体系进行了扩展。

在古教会斯拉夫语的音位系统上，格拉哥里字母与早期西里尔字母之间存在高度规则的对应关系。二者使用基本相同的字母命名体系，并表达同一套语音结构。虽然在个别历史字母与后期演化方面并非完全一一同构，但在大多数基础字母上可以建立稳定的映射关系。

示例：

```
Ⰰ  (Azъ)  ↔  А   /a/
Ⰲ  (Vědě) ↔  В   /v/
ⰳ  (Glagoli) ↔ Г /g/
```

因此，在技术实现中，可以将格拉哥里字母视为与西里尔字母之间存在系统性字母对应关系（systematic graphematic correspondence）。

### 3.2 理论基础 II：西里尔文到拉丁文的“学术转写”与 ISO 9 标准

将西里尔字母转换为现代英语所属的拉丁字母系统，在语言学中被称为**罗马化（Romanization）或转写（Transliteration）**。其理论依据主要建立在以下两大国际标准之上：

1. **学术转写（Scientific transliteration of Cyrillic）：** ^[3]^起源于1898年的《普鲁士编目规则》（Preußische Instruktionen），是斯拉夫语言学界通用的国际标准。该系统主要借助附加符号（Diacritics，如 č, š, ž）来确保西里尔辅音与拉丁字母的一一对应。
2. **ISO 9:1995 标准（Information and documentation — Transliteration of Cyrillic characters into Latin characters）：** ^[4-5]^国际标准化组织发布的严格转写系统。**关键结论：ISO 9 强调绝对的“单字符对单字符”（One character for one character）映射**，不考虑目标语言的发音习惯，仅为了确保文本能够100%无损地在西里尔与拉丁字母之间双向逆转。

但是，无论是学术转写还是 ISO 9，其目标语言都是“带有大量附加符号的扩展拉丁字母表”，而非“仅有26个字母的现代基础英语”。古斯拉夫语拥有超过40个音位，因此**在纯粹的语言学历史上，不存在直接映射到26个英文字母的完美等价表。**

## 5. 补充说明

为达成与26键英语键盘（ASCII基础拉丁区）的完全兼容，本转换模型包含以下几处显式的学术偏离（Deviations），特此声明：

1. **清软颚擦音与清喉擦音的妥协 (字母 H)：** 在标准语言学中，格拉哥里字母 `ⱈ / Ⱈ` (Xěrъ) 对应西里尔字母 Х，其国际音标（IPA）为 /x/（清软颚擦音，如德语的 *Bach*）。但是巫师三里面用的是这个。
2. **双唇鼻音符号的强行剥离 (字母 M)：** 古斯拉夫语中存在标准的双唇鼻音 /m/，其对应的格拉哥里字母本应为 `ⰿ / Ⰿ` (Myslite)。但根据巫师三的特殊设定，直接嵌入了现代拉丁字母 `m / M`。
3. **音系断层 (字母 Q 与 X)：** 英语字母 `q`（通常伴随 u 发 /kw/）与 `x`（发 /ks/）属于复合音或特定语言演化的产物。由于古斯拉夫音系中不存在完全等同的独立单字符（在西里尔文中 /ks/ 由 Ѯ 承担，但其已从现代字母表中淘汰且格拉哥里来源复杂），保留其拉丁字符本来面目。

## 6. 实践案例：多级转换路径演示

为了直观展示上述理论在实际文本中的应用，本节将通过一个具体的示范短语进行逆向解析（Decoding），清晰呈现 **格拉哥里字母** $\rightarrow$ **西里尔文** $\rightarrow$ **现代英语（拉丁文）** 的分步映射逻辑。

**初始文本（带自定义规则的目标格拉哥里序列）：** `Ⰰⱀⱅⰵⰴ Ⱅⱃⰻⱄⱄ Mⰵⱃⰻⰳⱁⰾⰴ`（第一节的triss通缉令）

### 步骤一：格拉哥里字母 $\rightarrow$ 西里尔文（字形还原）

根据底层音位同构原理，将格拉哥里字形逐一还原为历史西里尔文形式。

- **词元 1:** `Ⰰ` (А) `ⱀ` (н) `ⱅ` (т) `ⰵ` (е) `ⰴ` (д) $\rightarrow$ **Антед**
- **词元 2:** `Ⱅ` (Т) `ⱃ` (р) `ⰻ` (и) `ⱄ` (с) `ⱄ` (с) $\rightarrow$ **Трисс**
- **词元 3:** `M` (自定义保留) `ⰵ` (е) `ⱃ` (р) `ⰻ` (и) `ⰳ` (г) `ⱁ` (о) `ⰾ` (л) `ⰴ` (д) $\rightarrow$ **Mериголд**
  - *学术注释：* 注意此处首字母 `M` 触发了本系统的**自定义剥离规则**。它直接穿透了西里尔文的字形还原层级，未被还原为标准的西里尔字母 `М`。

### 步骤二：西里尔文 $\rightarrow$ 现代英语（罗马化转写）

应用前述的转写标准（参考发音等值映射），将西里尔序列转换为现代英语的26字母系统。

- **Антед** $\rightarrow$ **Anted**
- **Трисс** $\rightarrow$ **Triss**
- **Mериголд** $\rightarrow$ **Merigold**

**最终解析结果：** `Anted Triss Merigold`
